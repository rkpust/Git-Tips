# Git-Tips
Git Practices 


# User Email & Name 
- ### To Set User Email
  ```bash
  git config --global user.email "you@example.com"
  ```
- ### To Set User Name
  ```bash
  git config --global user.name "Your Name"
  ```
  
# Add File & Remove
For All File Replace FileName by .
- ### To Add File
  ```bash
  git add "FileName"
  ```
- ### To Remove File
  ```bash
  git restore --staged "FileName"
  ```
